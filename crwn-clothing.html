E-commerce - Retail
HTML | CSS/SASS | React + Redux | Firebase
Crwn Clothing Co. is an e-commerce urban clothing retail store built using React/Redux and SASS for the frontend and firebase for the backend database. The site features user registration and sign in using Google authentication, Stripe payment integration and cart dropdown.
